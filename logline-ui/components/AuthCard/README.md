# AuthCard

Cartão centralizado para autenticação LogLine ID.

## Props
| Prop     | Tipo        | Descrição                   |
|----------|-------------|-----------------------------|
| children | ReactNode   | Elementos a serem exibidos  |

## Uso

```tsx
<AuthCard>
  <NeonBrain status="idle" />
  <PasskeyButton>Entrar com Passkey</PasskeyButton>
</AuthCard>
```

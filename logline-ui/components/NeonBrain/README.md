# NeonBrain

Ícone animado do cérebro computável do LogLine ID.

## Props
| Prop   | Tipo   | Descrição                                  |
|--------|--------|--------------------------------------------|
| status | string | 'idle', 'registering', 'active', 'error'   |

## Uso

```tsx
<NeonBrain status="active" />
```
